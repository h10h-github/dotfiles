    class Guard::Notifier::Notifu

    System notifications using the [rb-notifu](github.com/stereobooster/rb-notifu) gem.

    This gem is available for Windows and sends system notifications to [Notifu](www.paralint.com/projects/notifu/index.html):

    @example Add the `rb-notifu` gem to your `Gemfile`

    group :development
  gem 'rb-notifu'
end

    @example Add the `:notifu` notifier to your `Guardfile`

    notification :notifu

    @example Add the `:notifu` notifier with configuration options to your

    `Guardfile` notification :notifu, time: 5, nosound: true, xp: true

def notify(message, opts = {})
  super
  self.class.require_gem_safely

  opts = DEFAULTS.merge(
    type:    _notifu_type(opts.delete(:type)),
    message: message
  ).merge(opts)

  # The empty block is needed until
  # https://github.com/stereobooster/rb-notifu/pull/1 is merged
  ::Notifu.show(opts) {}
end

def _notifu_type(type)
  case type.to_sym
  when :failed
    :error
  when :pending
    :warn
  else
    :info
  end
end
